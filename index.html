About
This page uses Mixitup with extensions Mixitup multifilter and Mixitup pagination. Mixitup is installed using `npm install mixitup` and declared in 'declare.d.ts'. Multifilter and pagination are local files added to the js folder.

Mixitup is used along with functions that manipulate the searchstring/URI in the browser - this way, one can reload the page without losing active filters!
